# Group Figure Composer / 组图工具

**Group Figure Composer** is a MATLAB toolkit for building publication-style
multi-panel scientific figures from MATLAB `.fig`, SVG, and raster image
inputs. It includes a Chinese interactive GUI and an English command-style
utility for GitHub/publication workflows.

**组图工具** 是一个 MATLAB 科研组图工具，用于把 MATLAB `.fig`、SVG 和常见位图素材组合成可投稿的多 panel 图。仓库同时包含中文版交互 GUI 和英文命令式轻量工具。

## Quick Start

```matlab
addpath(genpath(pwd))
```

Chinese interactive GUI:

```matlab
GroupFigureComposer
% or
GroupFigureComposerCN
```

English command-style utility:

```matlab
files = ["panel_a.fig", "panel_b.svg", "panel_c.png"];
GroupFigureComposerEN(files, "composite.png", ...
    Rows=1, Cols=3, WidthPx=1800, HeightPx=650, DPI=300);
```

Run the bundled English demo:

```matlab
run("examples/demo_lite.m")
```

## Main Features

- Fixed-pixel canvas sizing to keep preview/export geometry stable.
- Excel-like panel layout, panel merge/unmerge, manual panel movement and resizing.
- Per-panel `+` import buttons for `.fig`, `.svg`, `.png`, `.jpg`, `.tif`, `.bmp`.
- MATLAB FIG import with common axes style, legend, colorbar, colormap, and object data preserved.
- SVG parsing for common primitives, with optional Inkscape raster fallback for complex SVG files.
- Raster image auto-cropping to remove white margins.
- Unified font/font-size styling for axes tick labels, title, x/y/z labels, normal text, legend, and colorbar.
- Interactive selection/editing of axes and graphics objects through canvas clicks and the object tree.
- Dragging for panel labels, axes frames, text objects, rectangles, scale bars, and annotations.
- Snap guides for panel, axes frame, label, and annotation alignment.
- Automatic plot-frame alignment across rows/columns, including merged panels and the outer canvas perimeter.
- Palette swatches for recoloring lines, scatters, text, rectangles, annotations, and scale bars.
- Built-in scientific palettes and custom control-point colormaps with editable color-axis positions.
- Custom legend creation with explicit object selection.
- Text annotation at specified data coordinates.
- Project save/load for continuing edits later.
- Export to `.fig`, `.svg`, `.png`, `.tif/.tiff`, and `.pdf`.
- Data export to `.mat` as `groupData`, including panel map, source metadata, and extractable object data.

## Requirements

- MATLAB R2024b or newer is recommended.
- Inkscape is optional. It is only used as a fallback for complex SVG files.
